# P.F. Parrucchiere — Sito Vetrina

Sito demo per il salone P.F. Parrucchiere (Villorba, TV). Next.js + Tailwind CSS + Framer Motion.

- Hero a tutto schermo con parallasse e reveal del titolo
- Galleria lavori con scroll a velocità differenziate per colonna
- Sezione "Chi Siamo", linea prodotti, recensioni, contatti
- Marquee servizi e CTA con effetto shine

## Sviluppo locale

```bash
npm install
npm run dev
```
